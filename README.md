charity
=======
Toyanne Bennett
11/26/2013
Last Updated: 11/26/2013:


A ruby on rails application that lists charities, but sorted via zip code.


Functionality:
	*Add new zipcode
	*Add new charity
	*Pagination on homepage
Constaints:
	*There can be many charities to one zipcode, but only one zipcode to a charity.
